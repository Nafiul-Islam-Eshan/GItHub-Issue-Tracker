 - let has block scope. can be reassigned but can not be redeclared

### 3️⃣ What is the difference between map(), filter(), and forEach()?
- map()    --> Transform each element of an array and returns a new array
- filter() --> Keep only the elemets which satisfy the conditon and returns the array of matching elements
- forEach() --> Loop through each elements of array but returns nothing(undefined)
